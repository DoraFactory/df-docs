## Mainnet API endpoints

<Tabs items={['HTTPS', 'WebSockets']}>
  <Tab>
  ```text
  https://aptos.dorafactory.org/mainnet-api/
  ```
  </Tab>
  <Tab>
  ```text
  wss://aptos.dorafactory.org/mainnet-ws/
  ```
  </Tab>
</Tabs>

* Request method is `POST`.
* All requests need to submit a token in order to verify identity.
* All requests need to include the following Header: `Content-Type: application/json`
