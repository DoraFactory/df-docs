# Public Resources

There are several public resources available for development.

## Chain overview

We have now launched the Dora Vota Mainnet, identified with the chain-id: `vota-ash`.

## Genesis files

You can find the genesis file of the Mainnet `vota-ash` from our [GitHub repo](https://github.com/DoraFactory/doravota/blob/main/config/genesis.json).
