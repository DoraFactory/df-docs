# CosmWasm Contract

## CosmWasm on Dora Vota

[CosmWasm](https://cosmwasm.com/) is a library for writing Rust smart contracts. Dora Vota supports CosmWasm 1.0.0+ smart contracts.

## Install dependencies

Run the following command to download `rustup` and install Rust, then proceed by following the prompts:

```ansi filename="Terminal"
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Set up environment

<Steps>
### Set `stable` as the default toolchain channel

```ansi filename="Terminal"
$ rustup default stable
```

### Add WASM as the compilation target

```ansi filename="Terminal"
$ rustup target add wasm32-unknown-unknown
```

### Install `cargo-generate`, `cargo-run-script`, and `cosmwasm-check`

```ansi filename="Terminal"
$ cargo install cargo-generate --features vendored-openssl
$ cargo install cargo-run-script
$ cargo install cosmwasm-check
```

<Callout type="info">
  For more instructions, check out the [CosmWasm Starter
  Pack](https://github.com/CosmWasm/cw-template#readme).
</Callout>

</Steps>

<Steps>


### Store on Testnet

You can store the contract on either the Mainnet or the Testnet. In this example, we will demonstrate how to store on the Testnet:

```ansi filename="Terminal"
$ cd artifacts
$ dorad tx wasm store counter.wasm  --from <unsafe-test-key-name> --chain-id=<chain-id> --gas-prices 0.1uDORA --gas auto --gas-adjustment 1.3 -b block -y
```

- `<unsafe-test-key-name>`: Replace it with the key name from your local keys.
- `<chain-id>`: Replace it with `vote-ash` or `doravota-devnet`.

You will find a `CODE_ID` from the output of the command above. Save it as a local variable like `CODE_ID=XX`.

Instead, you can also run the following command to set the `CODE_ID` as a variable:

```ansi filename="Terminal"
$ TX=$(dorad tx wasm store counter.wasm  --from <unsafe-test-key-name> --chain-id=<chain-id> --gas-prices 0.1uDORA --gas auto --gas-adjustment 1.3 -b block --output json -y | jq -r '.txhash')
$ CODE_ID=$(dorad query tx $TX --output json | jq -r '.logs[0].events[-1].attributes[0].value')
$ echo "Your contract code_id is $CODE_ID"
```

Or, you can also look up the `code_id` on [Dora Vota Testnet Explorer](https://vota-testnet-explorer.dorafactory.org) by the storeâ€™s transaction.

### Instantiate the contract

```ansi filename="Terminal"
$ INITIAL_STATE='{"count":100}'
$ dorad tx wasm instantiate $CODE_ID $INITIAL_STATE --amount 50000uDORA  --label "Counter Contract" --from c1 --no-admin --fees 1uDORA --gas auto --gas-adjustment 1.3 --chain-id "doravota-devnet" --node https://vota-testnet-rpc.dorafactory.org:443
```

### Get the contract address

```ansi filename="Terminal"
$ CONTRACT_ADDR=$(dorad query wasm list-contract-by-code ${CODE_ID} --node https://vota-testnet-rpc.dorafactory.org:443 --output json | jq -r '.contracts[0]')
```

</Steps>

## Execute the contract

### Increment the count in contract

```ansi filename="Terminal"
$ INCREMENT_MSG='{"increment":{}}'
$ dorad tx wasm execute $CONTRACT_ADDR "$INCREMENT_MSG" --from c1 --gas-prices 0.01uDORA --gas auto --gas-adjustment 1.3 --chain-id "doravota-devnet" --node https://vota-testnet-rpc.dorafactory.org:443 -y
```

### Reset the count in contract

```ansi filename="Terminal"
$ RESET_MSG='{"reset":{"count":0}}'
$ dorad tx wasm execute $CONTRACT_ADDR "$RESET_MSG" --from c1  --gas-prices 0.01uDORA --gas auto --gas-adjustment 1.3 --chain-id "doravota-devnet" --node https://vota-testnet-rpc.dorafactory.org:443 -y
```

## Query the contract

### Get the contract state

```ansi filename="Terminal"
$ GET_STATE_MSG='{"get_count":{}}'
$ dorad query wasm contract-state smart  $CONTRACT_ADDR "$GET_STATE_MSG" --node https://vota-testnet-rpc.dorafactory.org:443
```

### Query the contract information

```ansi filename="Terminal"
$ dorad query wasm contract $CONTRACT_ADDR --node https://vota-testnet-rpc.dorafactory.org:443
```

## List all contracts on the blockchain

```ansi filename="Terminal"
$ dorad query wasm list-code --node https://vota-testnet-rpc.dorafactory.org:443
```
