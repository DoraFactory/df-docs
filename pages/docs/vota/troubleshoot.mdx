# Troubleshooting

## Installation of Node.js

Please check out the [Node.js official docs](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) for help.

## Problems with MACI round deployment

Report bugs with reproductions or screenshots to feedback@dorafactory.org.

## Problems with operator’s CLI commands

### Interruption during tally process

If your tally process is interrupted due to network issues, computer failure, or insufficient gas fees, please address and fix the corresponding issues and then re-run the tally command:

```bash npm2yarn
npm run tally
```

### Other issues

If you encounter other issues, please report them on [our GitHub repo](https://github.com/DoraFactory/vota-operator-scripts/issues). Our team will respond as quickly as possible. Your feedback is invaluable in helping us improve our scripts and processes.

## Questions or feedback about this manual

Any questions or feedback about this manual? Please feel free to [raise issues](https://github.com/dorafactory/df-docs/issues) or [post discussions](https://github.com/DoraFactory/df-docs/discussions) on our GitHub repo.
