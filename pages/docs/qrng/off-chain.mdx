## API reference

### `/random/GetRandomWords`

Get random numbers.

```bash
curl --request POST \
  --url $QRNG_API/random/GetRandomWords \
  --header 'Content-Type: application/json' \
  --data '{ "number": 1 }'
```

### `/random/GetCount`

Get the quntity of available random numbers.

```bash
curl --request POST \
  --url $QRNG_API/api/random/GetCount \
  --header 'Content-Type: application/json' \
  --data '{ }'
```
