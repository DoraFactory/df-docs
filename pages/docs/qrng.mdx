# QRNG Faucet

Dora Factory offers a faucet of QRNG (quantum random number generation) for scientific research and product testing. You can request device-independent quantum random numbers for free.

<Callout type="default">
  Learn more about the [device-independent random number
  generator](https://github.com/dorahacksglobal/quantum-randomness-generator).
</Callout>
